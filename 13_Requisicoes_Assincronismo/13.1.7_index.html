Fetch API

A Fetch API é uma API nativa do JavaScript que permite fazer requisições HTTP de forma assíncrona.

Ela substitui o uso de XMLHttpRequest (XHR) e oferece uma interface mais moderna e poderosa para lidar com requisições e respostas.

A Fetch API é baseada em Promises, o que torna o código mais legível e fácil de entender.

Ela é suportada por todos os principais navegadores modernos e é uma alternativa mais moderna e eficiente ao XMLHttpRequest.
