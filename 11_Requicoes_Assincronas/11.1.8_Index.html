Metodos de Envio de Dados

GET ou POST

GET

O método GET é usado para solicitar dados de um recurso específico. Ele é o padrão para requisições HTTP e é mais comum para operações de leitura.

POST

O método POST é usado para enviar dados para um recurso específico. Ele é mais comum para operações de escrita.

PUT

O método PUT é usado para atualizar um recurso específico. Ele é mais comum para operações de atualização.

DELETE

O método DELETE é usado para deletar um recurso específico. Ele é mais comum para operações de exclusão.
